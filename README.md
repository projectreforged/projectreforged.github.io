# Project Reforged

Project Reforged is an independent visual enhancement framework designed for the World of Warcraft® 1.12 client environment.

The project focuses on improving visual clarity, depth and atmosphere through advanced configuration and rendering techniques, including DXVK and ReShade — while preserving performance, stability and the original artistic identity of the game.

## Scope

- Visual enhancement through configuration and rendering optimization
- Advanced graphical refinement using external tools and presets
- Performance-aware visual improvements

This project is intended for personal use only.
