# Project Reforged

Project Reforged is an independent visual enhancement framework for the World of Warcraft® 1.12 client environment.

The project focuses on improving visual clarity, depth and atmosphere through curated integration and configuration of assets, combined with advanced rendering techniques such as DXVK and ReShade.
